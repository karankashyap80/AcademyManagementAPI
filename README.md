# 🎓 Academy Management API

A RESTful Web API developed using *ASP.NET Core, **C#, **Entity Framework Core, and **SQL Server* to manage academy student records. This project demonstrates CRUD operations, clean API design, and database integration.

---

## 📖 Project Overview

The Academy Management API is designed to manage student information efficiently. It allows users to create, read, update, and delete student records using RESTful API endpoints.

---

## ✨ Features

- ✅ Create Student
- ✅ View Student Details
- ✅ Update Student Information
- ✅ Delete Student
- ✅ SQL Server Database Integration
- ✅ Entity Framework Core
- ✅ RESTful API
- ✅ Swagger UI for API Testing

## 🚀 Getting Started

### Prerequisites

- Visual Studio 2026
- .NET 8 SDK
- SQL Server
- SQL Server Management Studio (SSMS)

### Installation

1. Clone the repository
2. Open the project in Visual Studio.
3. Update the SQL Server connection string in appsettings.json.
4. Run database migrations.
5. Start the application.
6. Open Swagger UI to test the API.

---

## 📡 API Endpoints

| Method | Endpoint | Description |
|---------|----------|-------------|
| GET | /api/Students | Get all students |
| GET | /api/Students/{id} | Get student by ID |
| POST | /api/Students | Add new student |
| PUT | /api/Students/{id} | Update student |
| DELETE | /api/Students/{id} | Delete student |
