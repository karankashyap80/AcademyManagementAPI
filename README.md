## 📌 About The Project

Academy Management API is a RESTful Web API built using *ASP.NET Core, **C#, **Entity Framework Core, and **SQL Server*. It enables efficient management of academy student records through secure and well-structured CRUD operations.

---

## ✨ Features

- ✅ Student Management
- ✅ CRUD Operations
- ✅ SQL Server Integration
- ✅ Entity Framework Core
- ✅ RESTful API
- ✅ Swagger API Documentation
- ✅ Clean Project Structure

## 🚀 API Endpoints

| Method | Endpoint | Description |
|--------|----------|-------------|
| GET | /api/Students | Get all students |
| GET | /api/Students/{id} | Get student by ID |
| POST | /api/Students | Create student |
| PUT | /api/Students/{id} | Update student |
| DELETE | /api/Students/{id} | Delete student |

## 💻 How to Run

1. Clone the repository.
2. Open the project in Visual Studio 2026.
3. Update the SQL Server connection string.
4. Run Entity Framework migrations.
5. Start the application.
6. Open Swagger and test the API.
