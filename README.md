# Academy Management API

## Project Overview
Academy Management API is a RESTful Web API developed using ASP.NET Core. It allows users to manage student records with CRUD operations.

## Features
- Add Student
- View Students
- Update Student
- Delete Student
- SQL Server Database
- Entity Framework Core
- Swagger API Documentation

## Technologies Used
- ASP.NET Core Web API
- C#
- SQL Server
- Entity Framework Core
- Swagger
